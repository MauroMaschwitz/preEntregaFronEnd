# MZ STORE

Página web propia orientada a la venta de productos electrónicos.

Actualmente el proyecto permite:
- Navegar por las diferentes secciones.
- Enviar consultas a través de un formulario de contacto.

Próximos pasos de desarrollo:
- Mostrar en cada sección los productos disponibles con sus características y precios.
- Actualizar semanalmente el recuadro **"Ofertas destacadas de la semana"** en el home, incorporando nuevas promociones.
